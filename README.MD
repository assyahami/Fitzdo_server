Fitzdo is a full-stack e-commerce web application built to demonstrate real-world product listing, cart flow, and backend API integration.
The project is structured with a separate frontend and backend, following scalable and production-ready practices.

====================================================Project Overview========================================================



This application allows users to:

1:Register
2:Login
3:Browse products
4:View product details

The backend provides REST APIs for products, users,auth while the frontend consumes these APIs and handles UI logic.

Tech Stack

Backend

Node.js
Express.js
MongoDB
Mongoose
JWT Authentication

Deployment

Frontend: Vercel

Backend: Render / Railway (or similar)

Database: MongoDB Atlas

====================================================Features========================================================

User Features

Product listing with pagination
Product detail page with image preview
Responsive design for mobile & desktop
Clean and simple UI
